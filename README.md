## 4. 👩‍💻 기능 목록 명세

- 홈 
  - 배너 캐러셀 구현
  - 홈화면에서 카테고리 아이콘 클릭시 카테고리별 브랜드 페이지로 이동
  - 땡처리콘 클릭시 구매 페이지 이동 
- 브랜드 페이지
  - 브랜드 아이콘을 누르면 각 해당 브랜드의 아이템이 뜸
  - 아이템 크릭시 구매 페이지로 이동
- 구매페이지
  - 해당 상품의 옵션을 선택할수있음 
  - 옵션 선택시 배경이 어두워짐
  - 옵션 취소 후 다시 선택가능
- 마이페이지
  - 홈에서 햄버거 버튼 클릭시 마이페이지로 이동 
  - 고객센터

## 5. 💿 설치 및 실행 방법

Project Clone

```bash
$ git clone https://github.com/wantedPreOnboarding/19_05th_shop/tree/develop
```

Project Setup

```bash
$ yarn install
```

Project Start For Development

```bash
$ yarn dev
```
